groundedwork: retrieval that knows when it doesn't know

A tiny, dependency-free retrieval layer that gives an LLM a knowledge base bigger than its context window, and says "I don't have that" instead of hallucinating. 2026-06-18

ContextForge: how to give a coding agent a codebase without drowning it in tokens

A companion to the workingset study, about a different engineer's open-source SDK optimized for code. I read the source, ran the 115 tests, and verified the mechanism: it ingests a codebase into a SQLite knowledge tree, retrieves a per-query working set via BM25, and generates a project file by file, compacting each finished file to its signatures so file 1,000 costs the same context as file 1. Includes the real 15-turn benchmark: 13.4× fewer tokens and 8× faster than an Azure Foundry Agent, with the turns the agent won shown too.
